Highlight #1 Github

Git is one of the main version control systems that is free and is an open source.Git is good because it is a distributed control system that allows every user to have a complete copy of repository data. Everything in Git is local and it has a very fast processing speed.

Highlight #2 What is Version Control?

Version control is one main repository for all of the files in a project. In this repository, co workers can check files out, makes changes, and then commit to the files.It helps with the common issue of file sharing where changes can get lost in a history of confusing names and date stamps.

Highlight #3 Version Control Benefits

Some benefits of a Version Control System, or VCS, are that the files cannot be overwritten, it's a common repository, or somewhere where files can be saved for all users, and it has a feature that allows people to work simultaneously Another cool feature is that VCS allows the user to revert back to an older version of their project.
